# Pricing Comparison Page

A simple responsive pricing comparison page built with **HTML** and **CSS** as part of my web development learning journey.

## 🚀 Live Demo

https://erfanmostafiz.github.io/pricing-comparison-project/

## ✨ Features

- Three pricing cards
- Featured "Pro" plan with gradient and glow effect
- Pricing comparison table
- Sticky bottom "Contact Us" bar
- Responsive layout using CSS Grid and Flexbox

## 🛠️ Built With

- HTML5
- CSS3

## 📚 What I Practiced

- Semantic HTML
- CSS Flexbox
- CSS Grid
- Positioning (`fixed`, `relative`, `absolute`)
- Gradients and box shadows
- Tables
- Hover effects
- Git & GitHub Pages deployment
